# Universidad-Web-Spring-Boot

Proyecto web desarrollado con Spring Boot para la gestión de universidades, rectores, selecciones y teléfonos.

## Características

- Gestión de Universidades
- Gestión de Rectores
- Gestión de Selecciones
- Gestión de Teléfonos
- Interfaz web con Thymeleaf

## Tecnologías

- Java
- Spring Boot
- Thymeleaf
- Maven

## Instalación

1. Clonar el repositorio
2. Ejecutar `mvn clean install`
3. Ejecutar `mvn spring-boot:run`
